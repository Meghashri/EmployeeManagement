Employee Management

There are following public JSON REST apis

# Route Method Type Full example route Description
1 /employees GET JSON http://localhost:5000/employees Get all employee data in paginated list
2 /employees_search/{id} GET JSON http://localhost:5000/employees_search/1 Get a single employee data with employee-id appended to url
3 /territory_employees/{territory_name} GET JSON http://localhost:5000/territory_employees/Australia Get all employee data in same country in paginated list
4 /lineofservice_employees/{line_of_service} GET JSON http://localhost:5000/lineofservice_employees/Tax Get all employee data in same Line of SErvice in paginated list
5 /employeelastname_search/{employee_lastname} GET JSON http://localhost:5000/employeelastname_search/Snow Get all employee data with same lastname in paginated list
6 /employees POST JSON http://localhost:5000/employees Add new employee data in JSON format(Using postman like client)
